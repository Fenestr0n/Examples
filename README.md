#  Git
### Клонирование репозитория

    cd Developer/
    git clone git@github.com:Fenestr0n/Examples.git

### Создание шаблона "Консольное приложение"

    cd Ex000/
    dotnet new console
    dotnet run

### Фиксация изменений
    cd ..
    git status
    git add Ex000/
    git commit -m "Hello, World!"

### Отправка изменений в удаленный репозиторий
    git push -u origin master
